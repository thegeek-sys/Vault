## for
In Go esiste un solo tipo di loop: il `for`. Nella sintassi non si ha nessuna parentesi `()` dopo il `for` statement ma sono sempre necessarie le parentesi graffe `{}`

```go
sum := 0
for i:=0: i &lt; i++ {
	sum += i
}
```
