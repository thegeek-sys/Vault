Data belonging to a single process is sent to all of the processes in the communicator

```c
int MPI_Bcast(
	void*        data_p,      // in/out
	int          count,       // in
	MPI_Datatype datatype,    // in
	int          source_proc, // in
	MPI_Comm     comm         // in
);
```

data_p: rank 0 invia valore tutti gli altri si trovano i dati ricevuti
