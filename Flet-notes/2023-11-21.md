## Pack/Unpack operator, funzioni con argomenti variabili
```python
*full_name, = 'iacopo', 'lizardking', 'masi'
full_name # -> ['iacopo', 'lizardking', 'masi']

*full_name, last = 'iacopo', 'lizardking', 'masi'
full_name, last # -> ['iacopo', 'lizardking'] 'masi'

```
