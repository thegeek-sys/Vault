in conclusione:
```java
for (<inizializzazione>; <espressione booleana>; <incremento>) {
	<istruzioni>;
}

// equivalente a
<inizializzazione>;
while (<espressione booleana>) {
	<istruzioni>;
	<incremento>;
}
```

Posso anche effettuare delle assegnazioni e incramenti multipli
```java
for (int k = 0, i = 0; i <= 10; i++, k+=5) {
	// codice iterazione
}
```
